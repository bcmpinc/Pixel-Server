Pixel-Server
============

A microhttpd c++ webserver that serves tiles (images) of an artificial 
landscape map.

Dependencies
------------
- libmicrohttpd
- libpng
- SCons
- g++

Running
-------
- Open a terminal
- Execute ``scons``
- Execute ``./pixel-server``
- Open a webbrowser and navigate to http://localhost:9999
